# 📊 HR Data Analysis with Python

A comprehensive data analysis project focused on extracting meaningful insights from HR datasets using Python. This project explores employee distribution, salary patterns, department-wise analysis, and workforce trends to support data-driven HR decisions.

---

## 🚀 Project Overview

This project analyzes HR data to answer key business questions such as:

* Employee status distribution (Active, Resigned, Retired, Terminated)
* Work mode insights (On-site vs Remote)
* Department-wise employee count
* Salary trends across departments and job roles
* Identification of high-paying job titles

The goal is to uncover patterns and insights that can help organizations improve workforce planning and decision-making.
